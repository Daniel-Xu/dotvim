## dotvim ##

Daniel-Xu's vim conf

## usage ##

* install vundle
`git clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`

* git clone
`git clone git@github.com:Daniel-Xu/dotvim.git`

* rename dotvim
`mv dotvim ~/.vim/`

* in vim buffer, `BundleInstall`

* in bash, `ln -s ~/.vim/vimrc ~/.vimrc`
